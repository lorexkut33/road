# 🐧 Справочник команд Linux

---

## 📁 Навигация и файловая система

| Команда | Описание |
|--------|----------|
| `pwd` | Показать текущую директорию |
| `ls` | Список файлов в директории |
| `ls -la` | Список всех файлов с правами и скрытыми |
| `cd /path` | Перейти в директорию |
| `cd ~` | Перейти в домашнюю директорию |
| `cd ..` | Перейти на уровень выше |
| `mkdir dir` | Создать директорию |
| `mkdir -p a/b/c` | Создать вложенные директории |
| `rm file` | Удалить файл |
| `rm -rf dir` | Удалить директорию рекурсивно |
| `cp src dst` | Копировать файл |
| `mv src dst` | Переместить / переименовать |
| `touch file` | Создать пустой файл |
| `find / -name "*.log"` | Найти файлы по имени |
| `du -sh dir` | Размер директории |
| `df -h` | Свободное место на дисках |

## 📄 Работа с файлами

| Команда | Описание |
|--------|----------|
| `cat file` | Вывести содержимое файла |
| `less file` | Постраничный просмотр |
| `head -n 10 file` | Первые 10 строк |
| `tail -n 10 file` | Последние 10 строк |
| `tail -f file` | Следить за файлом в реальном времени |
| `grep "pattern" file` | Поиск строки в файле |
| `grep -r "pattern" dir` | Рекурсивный поиск в директории |
| `wc -l file` | Количество строк в файле |
| `diff file1 file2` | Сравнить два файла |
| `chmod 755 file` | Изменить права доступа |
| `chown user:group file` | Изменить владельца файла |

## 📝 Bash — полезные конструкции

```bash
# Переменные
NAME="world"
echo "Hello, $NAME"

# Условие
if [ -f file.txt ]; then
  echo "Файл существует"
fi

# Цикл по файлам
for f in *.log; do
  echo "$f"
done

# Цикл по числам
for i in {1..5}; do
  echo "Итерация $i"
done

# Функция
hello() {
  echo "Hello, $1"
}
hello "Artём"
```
